HTML常用标签

表格<table>

  1. 表格基本属性：
             table         表格的开始和结束
         tr/th/td      行/列/单元格
         border        边框
         width        宽度
         bgcolour    背景色
         cellpadding 单元格和边框间距
         cellspacing 单元格之间的间距
  2. 表格的合并： 
             colspan 列
         rowspan    行
  3. 表格的页眉<thead>，主体<tbody>，页脚<tfoot>，表格的标题<caption>
  4. 表格对齐方式：
             {align|left,right,center}
             表格相对于周围元素的对齐方式。
         {valign|top,middle,bottom,baseline（与基线对齐}
             单元格中内容的垂直排列方式。
  5. 表格边框：frame
             viod    不显示外边框
         border    四周都显示边框
         above    显示上部外边框
         below    显示下部外边框
         hsides    显示上下外边框
         lhs        显示左部外边框
         rhs        显示右部外边框
         vsides    显示左右外边框
  6. 单元格边框：rules
             none    不显示内边框
         rows    只显示行边框
         cols    只显示列边框
         all        内边框全显示

表单<form>

  1. name/id：控件名称可重复/标签唯一标识
  2. 一般使用<label>文本</label>，便于对文本的控制
  3. 一般设置value属性以便于将收集的信息传到后台
  4. 输入型表单格式：input type
                 text        纯文本
             password    密码
             radio        单选（需设置name相同以实现
             checkbox    复选框
             button      仅在前端处理
             submit        到后端处理（提交
             hidden        隐藏的文本框
             file        提交文件
             checked        设置为true则为默认值
  5. 列表型表单格式：select option 
                 size        设置以展开列表
             selected    设置为selected则为默认值
  6. 大文本框：textarea
             rows      高度
         cols      宽度
