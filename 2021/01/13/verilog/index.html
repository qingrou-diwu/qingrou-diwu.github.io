第一章

概念

  1. HDL——硬件描述语言
  2. EDA——电子设计自动化
  3. ASIC——专用集成电路
  4. FPGA——现场可编程门阵列
  5. IP核——具有知识产权核的集成电路芯核的总称
  6. RTL——寄存器传输级
  7. System——系统级
  8. Algorithm——算法级
  9. Gate——门级
  10. Switch——开关级

数字电路设计流程

总体方案-系统建模-RTL编码-功能验证-综合-时序验证-物理综合/布局布线-物理验证-原型建立测试-工艺实现

第二章

语言要素

  • 命名信号名，模块名，参数名，可以是字母数字下划线$，区分大小写，首位必须字母或下划线。
  • 关键字都是小写，always是关键字，ALWAYS就不是，可以做标识符。
  • 0：低电平，假
    1：高电平，真
    x,X：未知
    z,Z,?：高阻态
  • 位宽未定义则以实际为准
    位宽大于实际则左侧补0
    位宽小于实际则左侧截断

数据类型

  • 连线类型：wire连线，tri三态线
  • 寄存器型（reg）：初始值不确定，位宽默认为1
  • 抽象数据类型：
  1. 整形：integer
  2. 时间型：time
  3. 实型（变量）：real（浮点数）
  4. 参数（常量）：parameter

运算符

  • 表达式的位宽由左侧决定
  • 逻辑等式运算符：== !=
  • 按位比较运算符：=== ！==
  • 逻辑运算与或非 && || ！
  • 按位运算取反，与，或，异或，同或 ~ & ^ ^~
  • 位数不同位运算右侧对齐左侧补0
  • 归约运算符（按位）：与，或，异或，相应非，每一位与后一位运算
  • 移位运算符：<< >>,左侧去掉右侧补0，右侧去掉左侧补0
  • 条件运算符，A？B：C A真则执行B,假则执行C
  • 连接运算{}：后面的信号依次连接前面的尾端
  • 复制运算{ { } } 复制运算:c = { 2 { a } },将a复制两次

模块

module 名字（信号1，信号2)
input 信号1；
output 信号2；
reg 信号2 ；// 被赋值必须定义reg类型
always@（posedge clk)
过程块；
endmodule

  • 端口缺省默认wire
  • 端口声明方式：input [位宽-1,0] 端口名字
